Smart Road Camera System (AI-Based Traffic Monitoring)

An AI-powered smart road camera system that uses computer vision and deep learning to monitor traffic conditions in real time. The project simulates a smart city traffic surveillance solution by performing vehicle detection, traffic analysis, speed monitoring, accident detection, and license plate detection.

Overview

This project demonstrates the use of deep learning and computer vision techniques for intelligent traffic monitoring. It processes video input to detect vehicles, analyze traffic density, estimate vehicle speed, detect accidents, and visualize results through a web interface.

Features

Vehicle detection and classification using YOLOv4 (Tiny)

Traffic density analysis

Speed limit monitoring

Accident detection

License plate detection

Flask-based web interface for visualization

Annotated video output generation
